# CHANGELOG - AE Kaitun (Expedition Script)

## 📝 ประวัติการแก้ไขและปรับปรุง Logic (Fix History)

### 1. แก้ไขปัญหายูนิตไม่ยอมวาง (`unit วางไม่ได้`)
- **สาเหตุ**: ฟังก์ชัน `buildAAStylePlaceCFrames()` ใน `src/PlacementEngine.lua` ถูกแก้ไขลดทอนตรรกะจนสแกนจุดจาก Anchor เดียว ทำให้ `hard = {}` เป็น 0 และไม่สามารถวางยูนิตได้
- **การแก้ไข**: 
  - ถอดแบบ `buildAAStylePlaceCFrames()` จาก `Kaitun.lua` มาใช้งานแบบ 1:1 ครบทั้ง 6 แหล่งสแกนพิกัด (มอนใกล้ฐาน, มอนทั่วไป, ปลายทางฐาน, ทางเดินมอน, GroundPlacement, และตำแหน่งตัวละคร)
  - เพิ่มระบบ **Emergency Fallback Scan** สแกน `GroundPlacement` ถี่ๆ เมื่อไม่พบจุดปกติ การันตีว่ามีจุดวาง CFrame เสมอ

### 2. ปรับ Logic การวางและการอัปเกรด (Farm vs Combat)
- **การวาง**: วางยูนิตผลิตเงิน (Stone Alchemist, Ramen Guy, Bulma ฯลฯ) ก่อนจนครบลิมิต จากนั้นวางยูนิตสายโจมตีโดยเรียงตาม **DPS/ดาเมจสูงสุดลงมา**
- **การอัปเกรด**: เน้นอัปเกรดตัวผลิตเงินจนเต็มก่อน หากเงินยังไม่ถึงค่าอัปเกรดตัวผลิตเงิน จะนำเงินไปอัปเกรดยูนิตโจมตีที่มี DPS สูงสุดที่พอซื้อได้ในระหว่างรอ

### 3. แก้ไขปัญหาช่อง Hotbar ถูกล็อก
- ปรับ `isSlotUsable()` ให้เช็คว่ามียูนิตสวมใส่อยู่จริงหรือไม่ เพื่อป้องกันไม่ให้โดนบล็อกด้วยไอคอนล็อกเลเวลบน UI เกม

### 4. ป้องกันการทำงานทับซ้อนในแต่ละโหมด (Gamemode Isolation)
- แยกแยะโหมดการเล่นด้วย `getMatchGamemode()` เพื่อให้ระบบเช็คด่านของ Story Mode ทำงานเฉพาะในโหมด Story เท่านั้น ป้องกันการสั่งย้อนกลับ Lobby หรือแทรกทับคิวเมื่อเล่นโหมด **Challenge** หรือ **Unit Trial**
